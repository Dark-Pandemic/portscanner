# portscanner
A high-performance Python port scanner designed for cybersecurity enthusiasts and professionals. Scans target hosts for open ports, identifies services, grabs banners, and generates detailed CSV/JSON reports.
Features
Core Scanning

Multithreaded port scanning for speed

Scan full port ranges or top common ports

Open port detection with service enumeration

Banner grabbing for extra info

Reporting & Logging

CSV export of scan results

Optional JSON export for integration

Scan summary including duration and open port count
